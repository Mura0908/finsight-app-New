FinSight Budget App

Uw persoonlijke financiële beheerapp

Let op: De APK is beschikbaar via GitHub Releases.

Download van GitHub

App Informatie

Versie: 1.0

Grootte: ~9.7 MB

Vereisten: Android 5.0 of hoger

Installatie stappen:

  1. Klik op de "Download van GitHub" knop hierboven
  2. Selecteer de nieuwste release en download het APK-bestand
  3. Ga naar je telefooninstellingen en schakel "Onbekende bronnen" in
  4. Open het gedownloade APK-bestand
  5. Volg de installatie-instructies op het scherm
  6. Start de app en begin met het beheren van je financiën!

Belangrijke opmerking

Deze app is gemaakt voor persoonlijk gebruik. Alle gegevens worden lokaal opgeslagen op uw apparaat en worden nergens anders naartoe verzonden.
